# Example Assets

These public images are tracked in the repository so Demo 2 can run after
cloning without private local files. They are small, qualitative inputs for
pretrained ViT inference, not a benchmark dataset or accuracy evaluation.

The public manifest is:

```text
examples/assets/manifest.txt
```
